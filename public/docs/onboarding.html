Onboarding

Welcome to Virtual Revelation

This guide gets first users into the system in minutes.

First user checklist

  1. Open the broker: /broker/.
  2. Choose a template and click Open Workspace.
  3. Open the wiki: /wiki/.
  4. Read the Welcome page and add a first note.
  5. Use the Share panel to invite another participant.

Roles

  • Observer: reads projections only.
  • Contributor: emits events via canvas/wiki/forum.
  • Operator: enables services and manages deployments.

Guiding rules

  • Events are append-only.
  • Projections are derived from events.
  • Transport is replaceable and not truth.
