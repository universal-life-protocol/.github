Quick Start

Virtual Revelation

A shared system that turns events into visual, spatial, and audio artifacts.

Start in 60 seconds

  1. Open the broker: /broker/.
  2. Pick a template and click Open Workspace.
  3. Open the wiki: /wiki/ and add a first note.
  4. Open the demo: /demo/.

What you will see

  • SVG + GLTF + audio artifacts from the same events.
  • Append-only records; nothing is overwritten.
  • Projections can be shared by link.
