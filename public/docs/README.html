Quick Start

Virtual Revelation

A shared, deterministic system for building worlds from events. No account needed.

Start here

  1. Open the broker: /broker/.
  2. Pick a template and click Open Workspace.
  3. Open the wiki: /wiki/ and add a first note.
  4. Share using the Share panel.

Rules

  • Everything is append-only.
  • Views are projections of events.
  • Transport is replaceable and not truth.

Where to explore

  • /canvas/ for spatial thinking.
  • /wiki/ for structured knowledge.
  • /forum/ for discussion threads.
  • /scene/ for 3D projections.
